## Bitstamp orderbook
Getting orderbook live data using golang & websockets for different usage such as data gathering, analytics, ...etc.

### Requirements
- go, go1.13.8
- nhooyr.io/websocket
```shell
    go get "nhooyr.io/websocket"
```

### Installation
```shell
cd /go-bitstamp-orderbook
go run main.go
```
